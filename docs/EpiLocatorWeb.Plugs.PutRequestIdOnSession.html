    EpiLocatorWeb.Plugs.PutRequestIdOnSession (epi_locator v0.1.0) View Source

    Plug that puts the request_id on the session.

    This is done so that a liveview can read the request_id value regardless of whether it's disconnected, i.e., a regular stateless connection, or connected, i.e., a websocket connection.
